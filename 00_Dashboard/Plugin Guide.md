# Plugin Guide

The vault works fully with **Obsidian core features only**. Start there.

## Core plugins to keep enabled

- Backlinks
- Bookmarks
- Command Palette
- File Explorer
- Outgoing Links
- Page Preview
- Properties View
- Quick Switcher
- Search
- Tags View
- Templates

The template folder is already configured as `99_Templates`.

## Optional community plugins

Install these only when a real need appears:

### 1. Obsidian Git — recommended later
Use it to back up and sync the vault to a private GitHub repository. Learn the basic manual vault first, then add Git.

### 2. Tasks — optional
Useful when you want filtered task views by tag or due date. This vault deliberately avoids depending on it.

### 3. Dataview — optional
Useful for automatic dashboards and note tables. Do not install it merely to make the vault look impressive.

### 4. Templater — optional
Useful for automatically inserting dates and filenames into templates. The built-in Templates plugin is enough initially.

## Avoid initially

Avoid installing themes, calendars, habit trackers, Kanban systems, complicated databases, and large plugin bundles. They recreate the same bulk that made Notion uncomfortable.
