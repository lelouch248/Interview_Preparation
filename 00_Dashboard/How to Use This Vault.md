# How to Use This Vault

## For learning a topic

1. Open the relevant file in `01_Topics`.
2. Select one unchecked concept.
3. Create a note in the matching `02_Notes` folder.
4. Add an example from your own work.
5. Solve one practical or interview problem.
6. Explain it without reading your note.
7. Mark it complete only after steps 4–6.

## For a real-life problem

Use [[99_Templates/Real-Life Problem Template]]. Focus on requirements, scale, constraints, failure modes, observability, security, cost, and trade-offs.

## For mock interviews

Create one file per round in `03_Mock Interviews/Rounds` using [[99_Templates/Mock Interview Template]]. Record weak answers immediately in [[06_Revision/Mistake Log]].

## For revision

Add only concepts you are likely to forget or repeatedly get wrong. The goal is a small high-value revision queue, not a copy of every note.
