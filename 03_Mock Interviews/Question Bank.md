# Mock Interview Question Bank

## SQL

- Explain the difference between `WHERE` and `HAVING`.
- Solve Top-N per group in two ways.
- Find consecutive active days for each user.
- Design SCD Type 2 SQL for late-arriving updates.
- Diagnose a join that unexpectedly multiplies rows.

## Snowflake

- Explain Snowflake architecture and virtual warehouses.
- Diagnose a slow query using Query Profile.
- Compare Streams + Tasks with Dynamic Tables.
- Design RBAC for engineers, analysts and service accounts.
- Reduce cost without harming pipeline SLAs.

## Python

- Build a retryable paginated API extractor.
- Explain generators and when they reduce memory pressure.
- Design testable code for a data validation framework.
- Compare threads, processes and asyncio.

## Data engineering design

- Design an incremental pipeline without a reliable primary key.
- Design CDC from SQL Server to Snowflake.
- Design a data-quality monitoring platform.
- Handle late events, duplicates and replay.
- Design lineage and impact analysis for warehouse objects.

## Behavioral

- Tell me about an automation you built.
- Describe a production failure and what changed afterward.
- Describe a disagreement over technical design.
- Explain a project where requirements were unclear.
- What would you improve in your current platform?
