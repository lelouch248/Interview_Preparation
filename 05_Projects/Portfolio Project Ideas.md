# Portfolio Project Ideas

Choose depth over quantity. Two polished projects are better than ten unfinished repositories.

## 1. Snowflake Environment Drift Control Tower

Compare objects across environments, classify differences, generate reports, track approvals, and produce safe remediation SQL.

## 2. Metadata-Driven Incremental Ingestion Framework

Ingest multiple source tables using configuration, checkpoints, batching, validation, retries, and restartability.

## 3. Data Quality and Observability Platform

Store reusable rules, execute checks, track freshness and volume, route incidents, and show product-level health.

## 4. Fivetran/HVR Operations Dashboard

Unify connector status, sync latency, errors, schema changes and ownership with alerting and runbooks.

## 5. Read-Only Agentic Snowflake Assistant

Use deterministic metadata checks plus an LLM to explain failures, drift or optimization opportunities. Require approval for any remediation.
