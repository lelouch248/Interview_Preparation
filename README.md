# Karthik's Interview Preparation Vault

A simple Obsidian vault for becoming interview-ready as a **Data Engineer / Software Engineer**.

This vault intentionally has:

- No weekly planner
- No monthly planner
- No complex productivity system
- No required community plugins
- One complete topic roadmap
- Dedicated notes folders
- Mock interview tracking
- Real-world engineering problem tracking
- Project and revision areas

## Open the vault

1. Extract the ZIP file.
2. Open Obsidian.
3. Choose **Open folder as vault**.
4. Select this folder.
5. Open [[00_Dashboard/Home]].

## The only workflow you need

1. Open [[00_Dashboard/Home]].
2. Choose a topic from [[01_Topics/00 Topic Index]].
3. Study one checklist item.
4. Create/update a note inside `02_Notes`.
5. Practise a problem or explain the concept aloud.
6. Mark the roadmap checkbox only when you can explain and apply it.

The system is designed to track **mastery**, not hours spent.
