# APIs & Backend Fundamentals

**Priority:** Supporting Skill

## Outcome

- Build and consume reliable APIs for data tools.
- Understand service boundaries, authentication and background work.

- [ ] Handle retries and duplicate requests
- [ ] Explain monolith vs services for an internal tool
